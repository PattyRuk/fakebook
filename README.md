# FakeBook Application

This web page will mimic a social network application that can post and view account informartion of the current user. The web page was created using HTML, CSS, and JavaScript. The post form accepts text and images, displays posts with the current time of posting.

## Features

*  Header with account information pop-up.
*  Form that submits text and/or images.
*  ES Module structure (`import`/`export`). With `User` (base) and `Subscriber` (derived) classes with private properties.
